# Projeto de Análise de Mercado e Estratégia - Programa de Trainees

## 📋 Visão Geral

Este projeto faz parte do **Programa de Trainees - Trilha Educacional** da Liga AgroTech, com foco no desenvolvimento de análises de mercado e estratégia no contexto do agronegócio brasileiro.

O programa prioriza o desenvolvimento de:

- **Pensamento estruturado**
- **Análise de mercado**
- **Raciocínio estratégico**
- **Comunicação executiva**


## 🎯 Objetivo

Desenvolver uma análise estruturada de um setor do agronegócio e de uma empresa inserida nesse contexto, evoluindo do entendimento de mercado para a análise de posicionamento competitivo.

O projeto serve como preparação para um case real em parceria com o Grupo Mantiqueira.


## 🌱 Macrosetores Disponíveis

Cada grupo deve escolher **um** dos seguintes macrosetores:

### 1. Sucroenergético
- **Escopo**: Cana-de-açúcar, etanol, açúcar e bioenergia
- **Pontos relevantes**: dinâmica de preços, produtividade agrícola, custos, políticas públicas
- **Exemplos**: Raízen, São Martinho, Atvos

### 2. Defensivos Agrícolas
- **Escopo**: Herbicidas, fungicidas, inseticidas e biológicos
- **Pontos relevantes**: inovação, regulação, ciclo de safra, canais de distribuição
- **Exemplos**: Bayer, Syngenta, BASF, Corteva

### 3. Fertilizantes
- **Escopo**: Fertilizantes minerais, organominerais e biológicos
- **Pontos relevantes**: custo de matérias-primas, logística, importação, adaptação agrícola
- **Exemplos**: Yara, Mosaic, Nutrien, Verde Agritech


## 📅 Cronograma e Entregáveis

### 📊 Entregável 1: Análise de Mercado, Drivers e Cadeia de Valor  
**Data**: 24/04

**Objetivo**: Compreender o setor e estruturar uma análise de mercado consistente

**Conteúdo**:
- Contexto do setor (tamanho, tendências)
- Identificação de drivers de mercado
- Cadeia de valor simplificada
- Análise inicial de uma empresa


### 📍 Entregável 2: Benchmark, Posicionamento e Oportunidades  
**Data**: 08/05

**Objetivo**: Analisar concorrência e posicionamento da empresa escolhida

**Conteúdo**:
- Benchmark (mín. 3 concorrentes)
- Análise de posicionamento
- Matriz de posicionamento
- Segmentação de clientes (alto nível)
- Identificação de oportunidades estratégicas
